# TAJRENT demo (mobile-first)

## Run locally

1. Open terminal in your cloned repository folder (your own local path, not `/home/runner/...`):

```bash
cd /path/to/your/korkovo.ab
```

2. Check you are in the right place:

```bash
pwd
ls
```

You should see `index.html` and `korkovo.ab`.

3. Start local server:

```bash
python3 -m http.server 4173
```

4. Open in browser:

```text
http://127.0.0.1:4173/
```

If you get 404, you started the server in the wrong folder.
